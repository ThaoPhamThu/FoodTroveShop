## Development

1. Clone the Repository and change directory

```
git clone https://github.com/ThaoPhamThu/FoodTroveShop

```

2. Install the dependencies.

```
cd frontend
npm install
```

cd backend
npm install

3. Run app locally.

```
cd frontend
npm start
```

cd backend
npm start

4. Deploy.

```
cd backend
docker build -t shop-backend .
docker container run -p 4141:4000  shop-backend

```
